# 유비쿼터스 언어 정의 (Glossary)

각 바운디드 컨텍스트에서 사용하는 핵심 개념과 상태를 변경하는 행위를 정의한다.

## 브랜드 컨텍스트

| 용어 | 영문 | 정의 |
|------|------|------|
| 브랜드 | Brand | 상품을 판매하는 주체. 이름을 가진다 |
| 브랜드 등록 | BrandRegistration | 관리자가 새로운 브랜드를 등록하는 행위 |
| 브랜드 수정 | BrandModification | 관리자가 브랜드 정보를 변경하는 행위 |
| 브랜드 삭제 | BrandDeletion | 관리자가 브랜드를 삭제하는 행위 |

## 상품 컨텍스트

| 용어 | 영문 | 정의 |
|------|------|------|
| 상품 | Product | 판매 가능한 항목. 이름과 가격을 가진다 |
| 재고 | Stock | 상품의 판매 가능한 수량 |
| 상품 등록 | ProductRegistration | 관리자가 새로운 상품을 등록하는 행위. 상품이 속할 브랜드가 사전에 등록되어 있어야 한다 |
| 상품 수정 | ProductModification | 관리자가 상품 정보를 변경하는 행위 |
| 상품 삭제 | ProductDeletion | 관리자가 상품을 삭제하는 행위 |

## 좋아요 컨텍스트

| 용어 | 영문 | 정의 |
|------|------|------|
| 좋아요 | Like | 사용자가 특정 상품에 관심을 표시하는 행위 |
| 좋아요 취소 | Unlike | 사용자가 좋아요를 철회하는 행위 |

## 주문 컨텍스트

| 용어 | 영문 | 정의 |
|------|------|------|
| 주문 | Order | 사용자가 상품 구매를 요청하는 행위 |
| 주문항목 | OrderItem | 주문에 포함된 개별 상품, 수량, 가격 |
| 주문취소 | OrderCancellation | 사용자가 주문을 철회하는 행위 |
