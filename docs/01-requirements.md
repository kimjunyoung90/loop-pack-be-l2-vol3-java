# 요구사항 정리

## 일반 사용자

### 브랜드
#### 브랜드 목록 조회
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 사용자는 브랜드 정보를 조회할 수 있다.
- 사용자는 최신 등록된 브랜드 순으로 조회가 가능하다.
- 사용자는 '좋아요'가 많은 순으로 브랜드 조회가 가능하다.
##### 유스케이스 흐름
- 로그인 여부 상관 X
- 브랜드 목록 정렬 조건 확인
- 조건별 브랜드 목록 정렬
- 브랜드 목록 조회
- 브랜드 목록 페이징 적용

#### 브랜드 정보 상세 조회
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 사용자는 특정 브랜드 정보를 조회할 수 있다.
##### 유스케이스 흐름
- 로그인 여부 상관 X
- 브랜드 정보 상세 조회

### 상품
#### 상품 목록 조회
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 사용자는 상품 목록을 조회할 수 있다.
- 사용자는 최신 등록된 상품순으로 상품 목록을 조회할 수 있다.
- 사용자는 가격 순으로 상품 목록을 조죄할 수 있다.
- 사용자는 '좋아요'가 많은 순으로 상품 목록을 조회할 수 있다.
- 사용자는 특정 브랜드의 상품 목록을 조회할 수 있다.
##### 유스케이스 흐름
- 로그인 여부 상관 X
- 정렬 조건 확인
- 정렬 조건별 상품 목록 정렬
- 상품 목록 조회
- 상품 목록 페이징 적용

#### 상품 정보 상세 조회
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 사용자는 특정 상품의 상세 정보를 조회할 수 있다.
##### 유스케이스 흐름
- 로그인 여부 상관 X
- 상품 상세 정보 조회

### 좋아요
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 사용자는 특정 상품에 대해 '좋아요'를 할 수 있다.
- 사용자는 자신이 '좋아요'한 상품에 대한 '좋아요'를 취소할 수 있다.
- 사용자는 자신이 '좋아요'한 상품 목록을 조회할 수 있다.
##### 유스케이스 흐름
- 로그인 여부 확인
- 상품 유무 확인
- '좋아요' 반영
- '좋아요' 여부 확인
- '좋아요' 취소
- '좋아요'한 상품 목록 조회

### 주문
#### 주문 생성
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 사용자는 상품을 주문할 수 있다.
##### 유스케이스 흐름
- 로그인 여부 확인
- 상품 재고 확인
- 상품 정보 확인(예: 가격)
- 상품 재고 확보
- 상품 주문
- 상품 재고 차감
- 상품 주문 완료

#### 주문 조회
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 사용자는 본인의 주문 내역을 조회할 수 있다.
- 사용자는 기간별 본인의 주문 내역 조회를 할 수 있다.
- 사용자는 특정 주문의 상세 정보를 조회할 수 있다.
##### 유스케이스 흐름
- 로그인 여부 확인
- 요청 기간 확인
- 본인 정보 확인
- 주문 목록 조회

## 관리자
### 브랜드

##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 관리자는 브랜드 정보를 등록할 수 있다.
- 관리자는 등록된 브랜드의 목록을 조회할 수 있다.
- 관리자는 특정 브랜드의 상세 정보를 조회할 수 있다.
- 관리자는 브랜드 정보를 수정할 수 있다.
- 관리자는 브랜드 정보를 삭제할 수 있다.
  - 브랜드 삭제 시 관련된 상품도 전부 삭제되어야 한다.
##### 유스케이스 흐름
- 관리자 여부 확인
- 브랜드 목록 조회
- 브랜드 정보 조회
- 브랜드 정보 등록
- 브랜드 정보 수정
- 브랜드 정보 삭제
  - 관련 상품 정보 삭제

### 상품
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 관리자는 상품 정보를 등록할 수 있다.
  - 단, 상품이 속할 브랜드가 등록되어 있어야 한다.
- 관리자는 등록된 상품 목록을 조회할 수 있다.
- 관리자는 상품 상세 정보를 조회할 수 있다.
- 관리자는 상품 정보를 수정할 수 있다.
- 관리자는 상품 정보를 삭제할 수 있다.
##### 유스케이스 흐름
- 관리자 여부 확인
- 상품 목록 조회
- 상품 정보 조회
- 상품 정보 등록
  - 상품이 속할 브랜드 정보 등록 여부 확인
- 상품 정보 수정
- 상품 정보 삭제

### 주문
##### 사용자 시나리오 & 행위 중심의 기능 목록 정의
- 관리자는 주문 목록 조회를 할 수 있다.
- 관리자는 특정 주문에 대한 상세 정보 조회를 할 수 있다.
##### 유스케이스 흐름
- 관리자 여부 확인
- 주문 목록 조회
- 주문 상세 정보 조회
